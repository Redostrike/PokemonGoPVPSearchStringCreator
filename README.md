# Pokemon Go PVP Search String Creator
Makes search strings based on user input.

Download the CSV files from PVPPoke to import the in this app. Other CSV won't work! PVPPoke does amazing stuff and be sure to support them!

I am in no form affiliated with PVPPoke this is mainly a tool to be used with the data they make available!

After upload you can filter what you want. (i personally use TOP 100).

When you click search string it wil be pasted in your clipboard with the full evolution line (so if blastiose is in there squirtle, and wartortle will also be in the filter list!)

In the menu string combinder you can make a combination string of great league and ultra league pokémon depending if you play both league's or not this can be handy to not have duplicates in your search string.
You can add your owe prefixes and suffixes the best to use is the &0attack&3-4hp&3-4defense and &age0 but you can add your own if you want.

Any questions be sure to ask.
